# 🅿️ Sistema de Estacionamento API

API completa para gerenciamento de estacionamento desenvolvida com Node.js, Express e Prisma, com documentação via Swagger.

---

## 🚀 Funcionalidades

- 👤 Cadastro de usuários
- 👥 Cadastro de clientes
- 🚗 Cadastro de veículos
- 🅿️ Gestão de vagas
- 🔄 Controle de movimentação (entrada e saída)
- 💰 Cálculo automático de valor por tempo estacionado
- 📊 Status de vagas (Livre / Ocupada)
- 📚 Documentação com Swagger

---

## 🛠️ Tecnologias

- Node.js
- Express
- Prisma ORM
- MySQL
- Swagger
- Dotenv
